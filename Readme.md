# Dependencies
- [node.js](http://nodejs.org/)
- [bower](http://bower.io/)
- [grunt-cli](http://gruntjs.com/)
- [compass](http://compass-style.org/)

# Build:
``
npm install
bower install
grunt
``

# Run:
Building for running is not needed. Application is fully build in webapp dir.
``
grunt connect
``
Open http://localhost:8080/ in browser.
